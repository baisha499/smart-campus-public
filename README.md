# 课程表小工具 - 使用说明

## 功能特点

### 独立组件窗口
- **天气组件**: 显示当前天气信息、温度、湿度等
- **今日课程**: 显示当天课程表，支持日期切换
- **倒计时**: 显示距离下一事件或结束的倒计时
- **当前活动**: 显示当前正在进行的活动和进度
- **提醒**: 显示系统通知和提醒事项

### 核心功能
- 所有组件窗口永远置于最上层
- 组件可独立控制开关
- 系统托盘图标，右键打开设置
- 支持课程表导入（JSON/CSV格式）
- 开机自启动设置

## 使用方法

### 安装依赖
```bash
npm install -g yarn #安装包管理器

yarn install
```

### 启动程序
```bash
yarn start
```

### 系统托盘操作
1. **右键托盘图标** - 显示设置菜单
2. **左键托盘图标** - 快速打开设置界面
3. **设置界面** - 控制各组件的显示/隐藏

### 设置选项
1. **组件显示控制** - 独立开关各个组件
2. **课程表管理** - 导入课程表文件
3. **系统设置** - 开机自启动和窗口置顶

### 课程表导入
支持两种格式：
- **JSON格式**: 标准的课程表数据结构
- **CSV格式**: 时间,科目,教室 的简单格式
- **txt格式**:更加方便地编辑、导入!


## 注意事项
- 所有组件窗口默认位于屏幕右侧，垂直排列
- 程序启动后只在系统托盘显示图标
- 设置界面可以通过托盘图标访问
- 所有窗口保持置顶状态


## 注意:导入的课表格式要如下
### json

```json
// 请按照以下的格式生成一校一周的课程表，包含星期一到星期五的课程安排，如果不是按照以下格式，程序将无法识别。
// 可以自己增加行数来添加其他课程。
{
  "星期一": [
    { "time": "08:00-08:45", "subject": "语文", "classroom": "301" },
    { "time": "09:00-09:45", "subject": "数学", "classroom": "302" },
    { "time": "10:10-10:55", "subject": "英语", "classroom": "303" },
    { "time": "11:10-11:55", "subject": "物理", "classroom": "304" },
    { "time": "14:00-14:45", "subject": "化学", "classroom": "305" },
    { "time": "15:00-15:45", "subject": "生物", "classroom": "306" }
  ],
 
}
```

注意：json文件不能有注释，所以要删除上面代码中"//”后的内容。

### txt

```bash
txt(1. 逗号分隔格式)
   1 星期一,08:00-08:45,语文,301
   2 星期一,09:00-09:45,数学,302
   3 星期二,08:00-08:45,英语,303
```

```bash
txt(2. 竖线分隔格式)
   1 星期一|08:00-08:45|语文|301
   2 星期一|09:00-09:45|数学|302
   3 星期二|08:00-08:45|英语|303
```
```bash
txt(3. 空格分隔格式)
   1 星期一 08:00-08:45 语文 301
   2 星期一 09:00-09:45 数学 302
   3 星期二 08:00-08:45 英语 303
```
```bash
格式要求：
   - 每行代表一门课程
   - 必须包含四个部分：星期、时间、科目、教室
   - 支持的星期：星期一、星期二、星期三、星期四、星期五、星期六、星期日
   - 时间格式：HH:MM-HH:MM（如：08:00-08:45）
```

### csv

```bash
 CSV格式要求：
   1 星期,时间,科目,教室
   2 星期一,08:00-08:45,语文,301
   3 星期一,09:00-09:45,数学,302
   4 星期二,08:00-08:45,英语,303
   5 星期二,09:00-09:45,物理,304

  格式说明：
   1. 第一行为标题行（可选）：包含"星期,时间,科目,教室"
   2. 每行代表一门课程，用逗号分隔四个字段
   3. 字段顺序：星期、时间、科目、教室
   4. 时间格式：HH:MM-HH:MM（如：08:00-08:45）
   5. 星期格式：星期一、星期二、星期三、星期四、星期五、星期六、星期日

  导入步骤：
   1. 打开设置界面
   2. 在"课程表管理"部分点击"选择文件"按钮
   3. 选择你的CSV格式课程表文件
   4. 系统会自动解析并导入

  注意事项：
   - CSV文件必须使用逗号作为分隔符
   - 系统会自动跳过标题行和空行
   - 每行必须包含完整的四个字段信息
   - 支持中文内容，建议使用UTF-8编码保存CSV文件
   ```
